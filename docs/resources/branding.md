# Branding Guidelines

Visual direction and copy tone for list, calendar, and map experiences.

## 1. Brand Direction
- Community-first directory for Breakfast Club meetups.
- Visual tone: refined, clear, human. Not a corporate dashboard.
- Playful but intentional. Cultural.

## 2. Product Feel
- Mobile-first and easy to scan.
- Minimal clutter. Large, obvious tap targets.
- UI should support quick planning and navigation.

## 3. Voice and Copy
- Concise, direct, practical.
- Prioritize accurate and current details over promotional language.
- Use clear fallback text when details are missing: `Contact host for more info`.

## 4. Typography Hierarchy
- Primary location/city labels carry strongest weight — large, bold, block.
- Metadata and scheduling details: secondary, muted, smaller.
- Utility actions: compact, pill-styled, legible.
- All-caps utility labels: small sizes only, used sparingly.

## 5. Color
- Base palette defined as CSS vars: `--paper`, `--ink`, `--line`, `--link`, `--rail`
- Night edition: `--night-sky`, `--night-sky-deep`, `--night-line`, `--night-ink`
- Flagship (Williamsburg): warm yellow/gold tint (`#fffbef → #fdf3cc`), gold border
- Map/action green: `#0d6e43`

## 6. Interaction Principles
- No hover-only dependencies for core actions.
- Controls must be reachable and readable on narrow screens.
- Avoid text/icon combos that interrupt content hierarchy.

## 7. Accessibility Baseline
- Preserve readable contrast and line spacing.
- Prevent horizontal overflow.
- Keep links and buttons large enough for touch.
- Do not hide key content behind decorative UI.
